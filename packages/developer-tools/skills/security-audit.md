# Security Audit Skill

This skill specializes in identifying security vulnerabilities, access control issues, and security anti-patterns in code and configuration files.

## Capabilities

- **Vulnerability Detection**: Identify common security issues like SQL injection, XSS, and authentication flaws
- **Access Control Review**: Analyze permission systems and authorization logic
- **Configuration Security**: Check for insecure configurations and hardcoded secrets
- **Dependency Security**: Review third-party dependencies for known vulnerabilities

## Usage

Use this skill when:
- Reviewing code for security issues
- Conducting security audits
- Implementing security best practices
- Preparing for security assessments
